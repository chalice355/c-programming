# 실습과제 2
## 매크로상수 NULL이 무엇인지 조사하라, C/C++언어에서 정의의 차이, 정의된 헤더파일, 사용용도 등
* NULL은 포인터가 아무것도 가리키지 않음을 나타내는 매크로 상수이다.
* C언어에서는 void 포인터 형태의 0으로, c++에서는 정수 0으로 정의한다.
* C언어에서는 <stdio.h>, C++에서는 <iostream>에 정의되어있다.
* 유효하지 않음을 명시하는 경우에 사용한다.

## 널문자와 NULL의 차이를 설명하시오, 정의, 자료형, 메모리크기, 실제값, 사용용도 등
* 널문자('\w')와 NULL의 정의는 각각 아스키코드의 0번 문자, 매크로 상수 0이다.
* 자료형은 char와 int또는 void*이다.
* 메모리 크기는 각각 0과 4또는 8이다.
* 실제값은 모두 0이며, 용도로는 각각 문자열의 끝을 표시, 포인터가 비어있음을 표시한다.

## 요즘은 NULL 대신에 nullptr을 사용하는데 nullptr은 무엇인지 조사하시오
* nullptr은 C++11부터 도입된 포인터 전용 리터럴이다. 기존 NULL이 정수 0으로 정의되어 발생하던 모호함을 해결하기 위해 만들어졌다. (형식의 안정성)

# 실습과제 3
```
int main() {
	cout << "입력할 정수의 개수를 입력하세요:";
	int n;
	cin >> n;
	if (n <= 0) return -1;
```
* 변수 n을 선언하고, 그 안에 입력을 정수값을 입력을 받는다
* n이 0보다 작은 경우(음수), -1을 리턴한다.
```
	int* p = new int[n];
	if (!p) {
		cout << "메모리를 할당할 수 없습니다.";
		return -1;
	}
```
* int 타입의 메모리 동적 할당
* p == 0(NULL)인 경우, 에러 메시지 출력 후 종료
```
	cout << n << "개의 정수를 입력 하시오." << endl;
	for (int i = 0; i < n; i++) {
		cout << i + 1 << "번째 정수: ";
		cin >> p[i];
	}
```
* 입력받은 정수만큼 반복문으로 반복하며 n개만큼 정수를 입력받아 배열에 저장
```
	int sum = 0;
	for (int i = 0; i < n; i++)
		sum += p[i];
	cout << "평균 = " << sum / n << endl;
	delete[] p;
}
```
* 누적 값을 저장할 sum을 선언 후 0으로 초기화
* n개의 정수를 배열 순서대로 누적합을 구하고 n만큼 나누어 평균값 산출
* delete[] p로 메모리 반환

```
int main() {

```
* n에 정수를 입력받는다 (0보다 작은 음수일 경우, -1 반환)
```
	double* p = new double[n];
	if (!p) {
		cout << "메모리를 할당할 수 없습니다.";
		return -1;
	}
```
* double 타입의 메모리 동적 할당
* p == 0(NULL)인 경우, 에러 메시지 출력 후 종료
```
	cout << n << "개의 실수를 입력 하시오." << endl;
	for (int i = 0; i < n; i++) {
		cout << i + 1 << "번째 실수: ";
		cin >> p[i];
	}
```
* 입력받은 정수(n)만큼 반복문으로 반복하며 n개만큼 실수를 입력받아 배열에 저장
```
	double max = 0;
	for (int i = 0; i < n; i++)
		if (max < p[i])	max = p[i];
	cout << "최대값은 " << max << endl;
	delete[] p;
}
```
* 최댓값을 저장하는 max 선언 및 0으로 초기화
* max보다 p[i]가 큰 경우. max에 저장
* delete[] p로 메모리 반환

# 실습과제 5
```
int main() {
	cout << "저장할 문자열의 크기를 입력하세요: ";
	int n;
	cin >> n;
	cin.ignore();
	if (n <= 0) return -1;
```
* 변수 n을 선언하고, 그 안에 입력을 정수값을 입력을 받는다
* n이 0보다 작은 음수일 경우, -1을 반환
* cin.ignore()로 개행문자 제거
```
	char* p = new char[n + 1];
	if (!p) {
		cout << "메모리를 할당할 수 없습니다.";
		return -1;
	}
```
* [n + 1]로 널문자 공간 포함
* p == 0(NULL)인 경우, 에러 메시지 출력 후 종료
```
	cout << "문자열을 입력 하시오: " << endl;
	cin.getline(p, n + 1, '\n');
	cout << "입력한 문자열은 ";
	for (int i = 0; p[i] != '\0'; i++)
		cout << p[i];
	cout << "입니다." << endl;
	delete[] p;
}
```
* [n + 1]만큼 공간 할당
* 공백문자 도달시 종료 / 공백문자 도달 전까지 출력
* delete[] p로 메모리 반환
