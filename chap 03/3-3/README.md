# 실습과제 1
## 객체의 생성과 소멸의 의미를 설명하라
* 객체의 생성은 클래스라는 설계도를 바탕으로 실제 데이터를 운영체제로부터 메모리를 할당을 받아 올리는 것을 말한다. 선언 한 순서대로 생성된다.
* 객체의 소멸은 프로그램 종료로 인해 더이상 필요가 없어졌을 때, 운영체제에 다시 메모리를 반납하는 것을 말한다. 선언 한 역순으로 소멸하며, 소멸 후에는 더이상 호출이 불가하다.

## 지역변수(지역객체), 전역변수(전역객체)의 차이를 조사하라, 선언위치, 접근범위, 생성시점, 소멸시점, 초기화 여부 등의 관점에서 비교하라
* 지역변수는 함수 또는 블록 내부에서 선언하며, 선언된 블록 내부에서만 접근이 가능하다. 지역변수의 이름이 같은 경우, namespace를 통해 명시적으로 접근이 가능하다. 생성 시점은 선언문에 도달한 순간 생성되고, 블록이 끝나는 순간 소멸한다. 지역 변수는 자동으로 초기화되지 않기 때문에 초기화하지 않을 경우, 쓰레기값(garbage value)이 들어간다.
* 전역변수는 프로그램 코드 최상단에 선언하며, 프로그램 전체에서 접근이 가능하다. 프로그램 시작 후, main함수 전에 생성되며, main함수 종료 후 프로그램 종료 전에 소멸한다. 전역변수는 0 또는 nullptr, 기본생성자 등으로 자동으로 초기화된다.

## 프로그램 실행 후 main함수가 실행되기 전에 어떤 일이 처리되는지 조사해보라
* 런타임을 초기화하여 스택, 힙, 표준i/o를 준비한다
* BSS를 초기화하여 초깃값이 없는 젼역변수를 0으로 채우거나 초깃값이 있는 전역변수의 값을 설정한다
* 전역 객체를 생성자 선언 순서대로 호출하여 생성한다

## main함수가 종료 후 프로그램이 종료하기 전에 어떤 일이 처리되는지 조사하라
* 전역 객체, 지역 객체 순으로 소멸자를 호출하여 소멸한다
* 버퍼를 비우고, 런타임을 정리하여 힙이나 스택 등의 자원을 반납한다
